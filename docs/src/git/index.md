## git使用碰到的问题

如果你在main分支新建一个分支并develop切换过去,如果你没有在develop分支修改文件,这两个分支看起来hash值是一样的,只有你修改了develop分支的文件并提交了代码才会不一样
这是你切换回去两个分支就会不一样
